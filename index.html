Motivation

Why create this page? I was interested in learning the Go Programming Language but didn't really have a project in mind when I set out to learn the language. I learn best if I use the language so I thought this might be a good approach. I could pick up the language while writing little code recipies for some common tasks. I figured I'd publish the page on the off chance that, a) I would carry through with the exercise, and b) that someone else might find it useful.

Acknowledgements

First, I'd like to acknowledge and thank the authors of the Go website for making their javascript and CSS available with an open license so that I could borrow a similar look and feel.

Again, due to the open nature of the Go website, a few of my code fragments are either directly copied or heavily influence my fragments below. Recall that the main motiviation of this site is for me to learn Go, so repeating a few examples is another way for me to pick it up.

Lastly, I'd also like to acknowledge that the overall idea of this format comes from several O'Reilly books I've purchased over the years. Namely, the [Language] Cookbook style of book. Those books assume the reader already has some basic idea of the language their purpose is to provide a functional reference companion to other "learing the lanaguage" material.
