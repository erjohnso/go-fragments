Motivation

Why create this page? I was interested in learning the Go Programming Language but didn't really have a project in mind when I set out to learn the language. I learn best if I use the language so I thought this might be a good approach. I could pick up the language while writing little code recipies for some common programming tasks. I figured I'd publish the page on the off chance that that someone else might find it useful and I could refer to it again in the future.

Acknowledgements

The overall idea of this material comes from several O'Reilly Cookbook style of books. Those books assume the reader already has some basic knowledge and experience with the the language and the primary purpose of the book is to provide a functional reference companion.

I'd also like to give advanced thanks to the authors of the Go website and community for the overall open (free) reference material. A few of the code fragments below are either directly copied or heavily influence my fragments below. Recall that the main motiviation of this site is for me to learn Go, so repeating a few examples is another way for me to improve my understanding.

And finally, I'd like to acknowledge and thank the authors of the Go website for making their javascript and CSS available with an open license so that I could borrow a similar look and feel.

About

These fragments were written using Go version 1 (1.0.3 to be exact). Unless noted otherwise, all fragments can be run by cutting and pasting into the playground, or by saving the file locally (e.g. to a file named example.go) and executing the following command in your shell/terminal:

$ go run example.go

Note: This assumes you have already installed Go.
